# Security Policy

## Supported Scope

This repository documents and automates a local operator workflow. Security-sensitive areas include:

- ADB access to Android devices
- SSH key handling
- Termux package installation commands

The latest `main` branch is the supported version.

## Reporting

Do not open a public issue for credential exposure, unsafe command guidance, or workflow steps that could leak private keys.

Report security concerns through GitHub Security Advisories or by contacting the repository owner directly through GitHub.

When reporting:

- describe the impacted step
- include reproduction conditions
- redact device identifiers, keys, and local paths that are not necessary
